# 🛍️ BagNest - Customer Storefront

The sleek, modern, and highly responsive frontend for BagNest customers. Designed with a luxury aesthetic, featuring dark-mode glassmorphism and smooth animations.

## 🚀 Features
- **Glassmorphism UI**: A premium, modern design system with frosted glass effects and mesh gradients.
- **Product Discovery**: Browse categories and view detailed product information.
- **Shopping Cart**: Fully functional cart with persistent state and real-time updates.
- **User Profiles**: Manage personal information and order history.
- **Responsive Design**: Optimized for mobile, tablet, and desktop experiences.
- **SweetAlert2 Integration**: Beautiful, interactive notifications for user actions.

## 🛠️ Tech Stack
- **Framework**: Next.js 15+ (App Router)
- **Styling**: Tailwind CSS
- **State Management**: React Context API (Cart & Auth)
- **Interactions**: SweetAlert2 & Lucide React

## 📦 Setup & Installation
1. Navigate to the user directory: `cd user`
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
