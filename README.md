# 🎒 BagNest - Full Stack E-Commerce Platform

BagNest is a high-end, full-stack e-commerce application for premium bags and travel gear. Built with the MERN stack (MongoDB, Express, React/Next.js, Node.js), it features a luxury dark-mode design and a robust management system.

## 📂 Project Structure
- `/admin`: Next.js dashboard for store management.
- `/user`: Next.js storefront for customers.
- `/server`: Node.js/Express API with MongoDB integration.

## 🛠️ Global Tech Stack
- **Frontend**: Next.js, Tailwind CSS, Axios, SweetAlert2.
- **Backend**: Node.js, Express.js, JWT, Bcrypt, Multer.
- **Database**: MongoDB (via Mongoose).

## 🚦 Getting Started
1. **Database**: Ensure MongoDB is running and update the `.env` in the `/server` folder.
2. **Server**: Run `npm install` and `npm run dev` in `/server`.
3. **Frontend**: Run `npm install` and `npm run dev` in both `/user` and `/admin`.
