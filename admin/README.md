# 👜 BagNest - Admin Dashboard

The administrative nerve center for the BagNest e-commerce platform. This dashboard allows administrators to manage the store's inventory, track customer orders, and oversee user activity.

## 🚀 Features
- **Product Management**: Create, update, and delete products with image upload support.
- **Order Tracking**: View and manage customer orders in real-time.
- **User Management**: Oversee registered users and their roles.
- **Analytics Overview**: View key metrics through a clean, responsive dashboard.
- **Authentication**: Secure login system for authorized personnel.

## 🛠️ Tech Stack
- **Framework**: Next.js 15+ (App Router)
- **Styling**: Tailwind CSS
- **State Management**: React Context API
- **HTTP Client**: Axios

## 📦 Setup & Installation
1. Navigate to the admin directory: `cd admin`
2. Install dependencies: `npm install`
3. Configure environment variables (API URL).
4. Run the development server: `npm run dev`
